API Magic

1. Создай волшебного героя

URL: POST {{hostname}}/user

Задание: Выбери тип в поле character — тебе доступны WizardMaleBlue и WizardFemaleFire. Придумай имя героя в поле firstName. Обязательные поля тела запроса: id, username, firstName, lastName, email, password, phone, userStatus, character. При верном типе персонажа и успешном ответе от бэка появится выбранный тобой волшебный герой. 

        


        
        

            
2. Отправка персонажа в сказочный мир

            
URL: GET {{hostname}}/user/{{username}}


            

                Задание: Укажи username в JSON-теле. Если сведения о персонаже на сервере совпадут с теми, что ты отправлял при создании (id/username/firstName/lastName/email/phone/userStatus), увидишь героя в волшебном месте. Если нет — покажем лягушку.
            


            

                

                    

                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
3. Открой Врата Святилища Тотемов

            
URL: GET {{hostname}}/user/login


            

                Задание: Укажи username и password в JSON-теле. Успешный ответ откроет вход в святилище выбора тотема.
            


            

                

                    

                        
                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
4. Создай своего тотема

            
URL: POST {{hostname}}/pet


            

                Задание: Заполни JSON питомца и выбери тотем в tags[0].name:
                catFire, catGreen, catPurple или catSnow.

                Суперсилы:
                
    
                    
  • catFire — Пылающий рывок: прожигает преграды, устойчив к огню.
    • 
                    
  • catGreen — Лесной лекарь: лечит союзников, маскируется в листве.
    • 
                    
  • catPurple — Теневой скачок: короткий телепорт, устойчив к контролю.
    • 
                    
  • catSnow — Морозный щит: замедляет врагов, оставляет скользкие лужи.
    • 
                

            


            

                

                    

                        
                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
5. Проверка создания тотема

            
URL: GET {{hostname}}/pet/{{ID_pet}}


            

                Задание: Укажи petId в path:
                {"path": {"petId": 15}}. При успешном ответе появится изображение выбранного тотема.
            


            

                

                    

                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
6. Вызов тотема

            
URL: POST {{hostname}}/store/order


            

                Задание: Заполни JSON заказа: id, petId,
                quantity, shipDate, status, complete.
                Успешный ответ — и ты увидишь тотема «в пути».
            


            

                

                    

                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
7. Проверка воссоединения тотема и персонажа

            
URL: GET {{hostname}}/store/order/{{ID_order}}


            

                Задание: Укажи orderId в path:
                {"path": {"orderId": 1}}. Если заказ найден и совпадает с выбранным тотемом, увидишь героя и тотема вместе.
            


            

                

                    

                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
8. Отмена выбора тотема

            
URL: DELETE {{hostname}}/store/order/{{ID_order}}


            

                Задание: Укажи orderId в path:
                {"path": {"orderId": 1}}. При успешном ответе тотем «растворится».
            


            

                

                    

                        
                        
                    

                    

                


                

            


            
            

        


        
        

            
9. Отмена выбора персонажа

            
URL: DELETE {{hostname}}/user/{{username}}


            

                Задание: Укажи username в path:
                {"path": {"username": "Barin555"}}. Успех — и персонаж исчезнет (картинка исчезновения).
