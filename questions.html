Нужен новый API‑ключ

Похоже, текущий ключ не работает. Получите бесплатный ключ в кабинете IO и вставьте его ниже.

Открыть страницу создания ключа
  1. Нажмите Create New API Key в правом верхнем углу.
  2. В форме укажите любое имя ключа, например QAtoDev.
  3. Оставьте значения как на скрине: Project — io.intelligence, Permissions — All, Expiration date — любой удобный срок.
  4. Нажмите Create API Key, затем скопируйте созданный ключ и вставьте его сюда.
