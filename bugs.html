Список багов для воспроизведения

Bug_1. Зум главной страницы при быстром нажатии кнопок тетриса

Предусловия:

Открыть песочницу на мобильном устройстве (Например: Iphone 13 Pro - iOS 18.2)

Шаги по воспроизведению:

1. Перейти в блок "Тетрис"
2. Нажать на кнопку "Вниз" множество раз (быстро)
3. Сместить фокус нажатия за область кнопки
4. Проверить зум страницы

Ожидаемый результат:

Масштаб страницы не изменился

Фактический результат:

Масштаб страницы увеличен

Bug_2. Отображение полей ввода по левому краю в десктопном размере экрана

Предусловия:

Открыть песочницу на десктопном размере экрана

Шаги по воспроизведению:

1. Перейти в блок "Практика Swagger"
2. Проверить отображение полей ввода

Ожидаемый результат:

Сортировка полей ввода по центру (десктопный размер экрана)

Фактический результат:

Сортировка полей ввода по левому краю (десктопный размер экрана)

Bug_3. Отсутствие функционала удаления двух и более заполненных полос в тетрисе

Предусловия:

Открыть песочницу на любом устройстве (Iphone 13 Pro - iOS 18.2)

Шаги по воспроизведению:

1. Перейти в блок "Тетрис"
2. Заполнить одновременно 2-х и более полос
3. Проверить сгорание всех полос одновременно

Ожидаемый результат:

При заполнении двух и более полос в тетрисе - они сгорают с начислением очков

Фактический результат:

При заполнении двух и более полос в тетрисе - сгорает только самая нижняя полоса с начислением очков

Bug_4. Отсутствуют склонения слова "дней" в блоке "Сколько осталось до 2025?"

Предусловия:

Открыть песочницу на любом устройстве (Iphone 13 Pro - iOS 18.2)

Шаги по воспроизведению:

1. Перейти в блок "Сколько осталось до 2025?"
2. Изменить число оставшихся дней
3. Проверить склонение для "21 день"
4. Проверить склонение для "4 дня"
5. Проверить склонение для "13 дней"

Ожидаемый результат:

Слово "дней" склоняется в зависимости от числа

Фактический результат:

Слово "дней" не склоняется в зависимости от числа

Bug_5. Ссылка под логотипом в хэддере использует общую стилизацию плашек на странице

Предусловия:

Открыть песочницу на любом устройстве (Десктоп Chrome 118.0.1)

Шаги по воспроизведению:

1. Навести фокус на иконку над названием "Тренировочный сайт"
2. Проверить наличие лишних элементов

Ожидаемый результат:

В области логотипа нет лишних элементов у логотипа

Фактический результат:

Ссылка под логотипом в хэддере использует серую плашку в качестве подложки
